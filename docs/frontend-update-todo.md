# Frontend Update TODO

本文档记录 MoviePilot Web / 后端更新中发现的、当前不属于 TV 端兼容必修项，但后续可以评估是否跟进的功能候选。

这些条目不是当前版本必须实现的兼容修复。每次执行 `.agents/prompts/frontend-update.md` 的上游更新分析时，应重新检查本文件：如果发现新的候选功能、已有条目已经不适合 TV 端，或某项已经变成兼容必修项，必须先在报告中说明建议变更，得到用户明确同意后才能更新本文件。

## 维护规则

- 只记录“可后续评估”的功能，不记录已确认必须修的兼容问题。
- 不因为 Web 前端新增功能就自动加入；必须说明它为什么可能适合 tvOS。
- 不把复杂后台管理能力默认列为 TV 端待办；除非它有明确的大屏使用场景。
- 每条 TODO 应包含来源、当前判断、TV 端需要评估的关键点。
- 更新本文件前必须先征得用户同意。

## 待评估功能

### 字幕搜索与下载

- 来源：MoviePilot Web / Backend `v2.13.6 -> v2.13.14` 新增字幕搜索、字幕结果展示和字幕下载相关能力。
- 当前判断：不是现有 TV 核心兼容必修项；TV 当前没有字幕搜索/下载入口。
- 可评估方向：
  - 是否在媒体详情页提供字幕搜索入口。
  - 是否在资源搜索结果页增加“字幕”模式或筛选。
  - 是否需要支持字幕下载保存路径选择。
  - 字幕下载属于副作用操作，若实现必须加入显式副作用兼容测试开关。
- 相关上游接口/路径：
  - Backend：`/search/subtitle/title`、`/search/subtitle/title/stream`、`/search/subtitle/media/{mediaid}`、`/download/subtitle`
  - Frontend：`SubtitleCard.vue`、`SubtitleItem.vue`、`AddSubtitleDownloadDialog.vue`

### Agent Assistant / Web Agent

- 来源：MoviePilot Web / Backend `v2.13.6 -> v2.13.14` 新增 Agent Assistant、会话历史、附件上传、音频输入输出等能力。
- 当前判断：偏 Web 交互和后台助手能力，不属于 TV 端兼容必修项。
- 可评估方向：
  - 是否存在适合遥控器/语音输入的大屏助手入口。
  - 是否只提供只读问答，避免在 TV 端执行复杂后台操作。
  - 是否需要复用后端 `AI_AGENT_ENABLE`、`AI_AGENT_HIDE_ENTRY`、音频能力开关。

### 通知中心与通知历史

- 来源：MoviePilot Web / Backend `v2.13.6 -> v2.13.14` 强化通知中心、通知历史和未读提示。
- 当前判断：TV 端已有本地错误通知机制，未实现完整通知中心。
- 可评估方向：
  - 是否需要一个只读通知列表，展示后台任务、订阅、下载结果。
  - 是否需要未读角标或系统页入口。
  - 是否避免把后台管理消息流直接搬到 TV 首屏。

### 下载任务高级字段与控制

- 来源：Backend `DownloaderTorrent` 新增 `save_path`、`content_path`、`category`、限速、trackers 等字段；Web 增加下载器后缀和更多下载控制。
- 当前判断：新增字段向后兼容，TV 当前下载列表可以继续忽略未知字段。
- 可评估方向：
  - 下载列表是否展示保存路径、分类、tracker 或限速信息。
  - 是否增加限速、标签、后缀等高级操作。
  - 是否保持 TV 下载页轻量，避免变成完整下载器管理后台。

### 订阅日历与特殊季显示

- 来源：Web 在订阅日历、特殊季标签、订阅进度展示上有持续优化。
- 当前判断：目前不是 TV 端兼容阻断；订阅核心契约仍由 `docs/subscription-compatibility-checklist.md` 维护。
- 可评估方向：
  - 是否增加 TV 端订阅日历视图。
  - 是否改进第 0 季、特别篇、特殊季标签显示。
  - 是否在订阅卡片上展示更细的入库/缺失进度说明。

## 暂不作为 TV 功能待办

以下能力目前默认不进入 TV 功能 TODO；除非用户明确要求或后续发现明确的大屏使用场景：

- 插件市场 Release 安装、Wiki 同步、插件日志下载、插件 doctor。
- 站点、用户、工作流、通知渠道、插件配置等复杂后台管理页面。
- Web 主题、动画、移动端布局、PWA 安装提示等纯 Web 体验优化。
